# NEXUS — Premium Electronics E-Commerce Platform

A fully functional single-page e-commerce application with AI-powered product recommendations, built with HTML, CSS, and Vanilla JavaScript.

![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)

---

## Features

- AI Shopping Assistant powered by REST API
- Single Page Application with custom routing
- Live search with instant dropdown results
- Cart with add, remove, and quantity control
- Wishlist with badge count and dedicated page
- User Signup, Login, and Logout
- Filter by category, price range, and rating
- Buy Now button with direct checkout flow
- Contact page with form and info cards
- Fully responsive design for all screen sizes

---

## Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **AI Integration:** REST API for shopping assistant
- **Storage:** Browser localStorage
- **Icons:** Font Awesome 6.5
- **Fonts:** Syne, DM Sans (Google Fonts)

---

## Project Structure

```
├── index.html    — All 6 pages in one HTML shell
├── styles.css    — Full design system with dark luxury theme
├── data.js       — 20 products across 8 categories
└── app.js        — All logic: routing, cart, auth, AI, filters
```

---

## Pages

- **Home** — Hero section, categories, featured products, deal banner, new arrivals
- **Shop** — All products with sidebar filters and sort options
- **Deals** — Discounted products with offer badges
- **Wishlist** — Saved products with remove option
- **Contact** — Contact form, phone, email, address, social links
- **Product Detail** — Full specs, Buy Now, Add to Cart, related products

---

## How to Run

```bash
git clone https://github.com/ParimiPujitha/E-commerce-Website.git
cd E-commerce-Website
```

Open `index.html` directly in your browser. No server or build tools required.
